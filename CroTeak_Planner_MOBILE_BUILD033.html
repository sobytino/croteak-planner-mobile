CroTeak cloud login
Prijavljen-
Session je spremljen lokalno. Ne spremamo lozinku kao običan tekst; app koristi Supabase session/refresh token.
Email confirmation ostaje uključen. Nakon potvrde maila vrati se i klikni Login.
Napredno: Supabase URL / key
Upute za prvi test
1. Ovdje upiši isti email/lozinku kao na PC-u i klikni Login.
2. Klikni Sync sada / Refresh za prvo povlačenje podataka.
3. Nakon toga samo dodaj ili promijeni posao i klikni Spremi.
4. Mobitel automatski šalje promjene u cloud. Gumb ↻ gore desno je brzi refresh.
