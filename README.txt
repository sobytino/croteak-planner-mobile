CroTeak Planner MOBILE BUILD032 PWA

Pokretanje:
1. Otvori index.html na mobitelu ili računalu.
2. Na Windowsu možeš kliknuti START_CROTEAK_MOBILE.bat.
3. Podaci se spremaju lokalno u browseru.
4. Za prijenos podataka koristi Meni -> Backup JSON i Import JSON.

Novo u BUILD032:
- Popravljena mobilna širina / header više ne bi smio bježati iz ekrana.
- PWA mobilni layout ostaje unutar širine mobitela.
- Dodan tab Poruke / Chat.
- Možeš ručno zalijepiti Facebook, Instagram ili WhatsApp chat.
- Chat se može povezati s poslom/brodom.
- Global Search traži i po spremljenim porukama.
- Posao ima detaljan pregled financija, chat i izvještaj.
- Backup JSON, Import JSON i CSV export.

Napomena:
Ovo još nije prava Facebook API integracija. Ovo je sigurna ručna faza: copy/paste chata u Planner. Prava integracija bi trebala Meta Developer app i online webhook server.
